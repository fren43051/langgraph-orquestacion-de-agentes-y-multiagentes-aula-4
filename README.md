# 2255 - LangGraph: Orquestación de Agentes y Multiagentes

Este repositorio contiene los materiales y scripts del curso **LangGraph: Orquestación de Agentes y Multiagentes** ofrecido por Alura Latam. El proyecto tiene como objetivo enseñar cómo diseñar e implementar sistemas avanzados de agentes inteligentes utilizando el marco LangGraph.

## Características Principales

- Ejemplos prácticos de orquestación de agentes.
- Uso de técnicas de streaming y persistencia de estados.
- Integración con herramientas de búsqueda y modelos de lenguaje avanzados.
- Implementación de flujos de trabajo secuenciales y condicionales.

## Requisitos del Sistema

- Python 3.x
- Entorno virtual (se recomienda `venv` o `conda`)
- Paquetes descritos en el archivo `requirements.txt` (si aplica)

## Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/Alura-Latam/langgraph-orquestacion-de-agentes-y-multiagentes-aula-4.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd langgraph-orquestacion-de-agentes-y-multiagentes-aula-4
   ```
3. Crea y activa un entorno virtual:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # En sistemas Unix o macOS
   .venv\Scripts\activate   # En Windows
   ```
4. Instala las dependencias necesarias:
   ```bash
   pip install -r requirements.txt
   ```

## Uso

El proyecto incluye un script de ejemplo para interactuar con los agentes. Para ejecutarlo, usa:

```bash
python main.py
```

También puedes explorar los ejemplos en los archivos `.ipynb` incluidos en el repositorio.
